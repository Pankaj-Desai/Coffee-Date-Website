# ☕ Coffee Date Proposal — for Mohini

> A cute, interactive proposal page built with pure HTML, CSS & JavaScript.  
> Because some questions deserve more than just a text message. 🥺❤️

---

## 💡 What is this?

This is a fun little web page Pankaj built to ask Mohini out on a coffee date — with a twist. Instead of a boring message, it features a **runaway No button**, a **growing Yes button**, and a full **date booking flow** so Mohini can pick the perfect time for their date. 😄

---

## ✨ Features

- **Side-by-side Yes / No buttons** — both visible and aligned on load
- **Escaping No button** — runs away on hover or tap, so saying No is harder than it looks 😂
- **Growing Yes button** — gets bigger and more tempting with every failed No attempt
- **Funny teasing messages** — a new one every time the No button escapes
- **No attempt counter** — keeps a running tally of how hard she tried to avoid it 😏
- **No button resignation** — after enough attempts, the No button gives up entirely and disappears
- **Celebration page** — floating hearts and a big YAYYYY when she finally says Yes ❤️
- **Date picker** — a full custom calendar to pick the date, plus time slots from 10 AM to 6 PM
- **Summary card** — a shareable recap card showing the confirmed date, time, and (most importantly) how many times she pressed No 😜
- **Copy & share** — one tap copies all the details so Mohini can send them straight to Pankaj 💌
